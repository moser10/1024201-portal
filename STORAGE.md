# 存储与容量说明

## Cloudflare 免费额度（2025–2026 常见政策，以控制台为准）

| 产品 | 免费额度 | 适合用途 |
|------|----------|----------|
| **R2** | 约 **10 GB·月** 存储；Class A 100 万次/月、Class B 1000 万次/月；**出站流量免费** | 附件、作品图 |
| **D1** | 约 **5 GB** 存储；读 500 万/天、写 10 万/天 | 元数据、用户记录 |
| **Workers** | 每天 10 万请求（免费层） | API、上传入口 |

### 能存多少人？（粗算）

假设人均占用 **25 MB**（中转站附件 + 偶尔一张作品图）：

- R2 10 GB ÷ 25 MB ≈ **400 活跃用户** 同时占满配额  
- 若人均仅 **5 MB**（只传小图/文档）≈ **2000 人**  
- 作品展示公开图可被反复浏览，R2 **不收出口流量费**，适合「展示」场景

D1 只存文件索引（文件名、大小、路径），**不是**文件本体；10 万条文件记录只占几十 MB 级。

```bash
cd ~/CodeProjects/1024

npm run deploy
```

验证：`curl -s https://1024201.com/api/health` → `"hasFiles": true`

API Token 需包含：**Account → R2 → Edit**。

## 中转站附件规则

- 槽位：**中转 1、中转 2（文本）+ 附件（原第 3 框）**
- 单文件 **20 MB**，每用户附件槽最多 **12** 个文件
- 展示：≤3 缩略图 · 4–6 图标 · 7+ 首图 + 数量

---

## 作品展示

- 仅图片；上传前在浏览器加**平铺水印** + 可选**时间戳**
- 时间戳按 **上传 IP 推断的时区**；使用 VPN/代理时可能不准（页面已提示）
- 单张最大 **20 MB**；发布后可分享 `view.html?id=...` 链接

## 个人云盘（100–200 MB / 人，7 天按账号滚动清空）放哪？

| | **Cloudflare R2** | **你的 VPS（60 GB）** |
|--|-------------------|----------------------|
| 适合 | 公开展示、全球 CDN、免流量费 | 私有盘、大配额、自定义清理逻辑 |
| 每用户 200 MB | 60 GB VPS ≈ **300 人** 满配额；R2 10 GB 免费 ≈ **50 人** | VPS 更宽裕 |
| 7 天按用户过期 | 需 Worker Cron + D1 记录 `expires_at`，逐对象删 R2 | **更推荐**：cron + 本地目录 `uid/` + `find -mtime` 或 SQLite |
| 运维 | 无服务器、自动扩展 | 需自己管磁盘、备份、安全 |

**建议：**

1. **作品展示 + 中转站附件** → 继续用 **CF R2**（已接入）  
2. **「每周清空」的个人云盘（100–200 MB）** → 放 **VPS**  
   - 60 GB 空间更适合多用户各 200 MB  
   - 按 `uploaded_at + 7d`  per-user 删除，不会一刀切  
   - 可用 MinIO / 简单 Go 或 Node 服务 + nginx 反代  

若坚持全放 CF：可用 R2 + D1 存 `expires_at`，Cron 每天扫过期对象；免费 10 GB 下人均 200 MB 只能约 **50 个满配额用户**，超出需买 R2 存储。
